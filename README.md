## Описание проекта

Реализовала:

* Список стран
* Детальную информацию о стране
* Кеширование
* Навигацию назад с экрана детальной информации
* Отображение соответсвующего эрана при отсутствии данных (чтобы увидеть, сделайте первый запуск без интернета)
* Отображение прогресс-индикатора во время выполнения запроса в сеть

При неудачном запросе в сеть отображаю диалог с двумя кнопками: повторить запрос и OK. 
При нажатии на OK диалог закрыватся, при нажатии повторить запрос делаем повторный запрос к серверу.

Многомодульная архитектура app, core и feature модули.

Сохранение данных в локальную бд при помощи библиотеки Room, кеширование картинок засчет ипользования бибилотеки Coil ->
Приложение имеет возможность работать без  интернета.

Деление feature модуля на слои data, domain, presentation.

Запросы в сеть с помощью Retrofit, так же подключено логирование.

### • Что знаете про HashMap / HashSet?

HashMap и HashSet — это коллекции, основанные на хешировании, обеспечивающие быстрый поиск, добавление и удаление элементов, 
но без гарантии порядка: HashMap хранит пары ключ-значение, а HashSet — только уникальные элементы (используя их как ключи во внутреннем HashMap)

---

### • Какие основные вещи знаете про корутины?

Корутины — предназначены для написания асинхронного кода.

В Kotlin корутины — на данный момент это основной инструмент асинхронного программирования, особенно в Android-разработке.

Suspend функции можно вызывать только в других suspend функциях или внутри корутин.
